## TradeLink Backend (MVP)

Backend API أولي لنظام ربط الموردين وتجار الجملة والتجزئة والمستهلكين.

### المتطلبات

- Node.js 18+
- قاعدة بيانات PostgreSQL مع قاعدة بيانات باسم `tradelink` (أو حسب متغيرات البيئة).

### الإعداد

1. تثبيت الحزم:

```bash
npm install
```

2. إنشاء ملف إعدادات:

انسخ `.env.example` إلى `.env` وعدّل بيانات قاعدة البيانات:

```bash
cp .env.example .env
```

3. تشغيل قاعدة البيانات (تأكد من تشغيل PostgreSQL).

4. تشغيل الخادم في وضع التطوير:

```bash
npm run dev
```

سيفتح الخادم على المنفذ `4000` افتراضياً.

### نقاط الـ API الأساسية

- `GET /` فحص أن الـ API تعمل.
- `GET /api/users` عرض المستخدمين.
- `POST /api/users` إنشاء مستخدم جديد.
- `GET /api/products` عرض المنتجات.
- `POST /api/products` إنشاء منتج جديد.
- `GET /api/orders` عرض الطلبات مع العناصر.
- `POST /api/orders` إنشاء طلب جديد مع عناصره.

يمكنك لاحقاً توسيع هذا الـ MVP لإضافة باقي الوحدات (الشحن، المدفوعات، التحليلات، إلخ) وفق المستند الذي أرسلته.
